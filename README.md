
# 🎓 Student Performance Predictor

## 📌 Overview
This project predicts student performance using Machine Learning.

It analyzes key factors like:
- 📚 Study Hours
- 📊 Attendance
- 📝 Previous Marks

and predicts the final score of a student.

---

## 🚀 Features
- Predict student performance instantly
- Simple and user-friendly interface
- Machine Learning model (Linear Regression)
- Fast and accurate predictions

---

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
